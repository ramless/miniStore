Shop Single Page Application

Проект использует React, Redux Toolkit, React Router, Axios.

- Просмотр списка товаров с загрузкой из API.
- Добавление товаров в корзину (с возможностью увеличения количества).
- Удаление товаров из корзины.
- Дропдаун корзины в хедере, где отображаются добавленные товары.
- Страница корзины с возможностью редактирования.
- Сохранение корзины при обновлении страницы с использованием localStorage
- Отдельные страницы для просмотра деталей продукта
- Форма checkout

Хуки
- useEffect
- useState
- useMemo
- useSelector
- useDispatch
- useLocation
